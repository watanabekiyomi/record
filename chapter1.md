# consoleを開こう
- Chrome の右上のその他ボタンをクリックし、出てきたメニューの中の「その他のツール」にマウスカーソルを合わせると、その左にさらにメニューが出てきます。
その中から「デベロッパーツール」を選択します。

# for文

# 配列
var i = [A, B, C, D];
- 配列の添字は0から始まる 
    - ex)i[0] = A


# 関数
function 関数名 (a){
    処理;
    return b;
}
- a:引数
- b:戻り値

# オブジェクト
var student =  {  
    name: '太郎',  
    age: 15 };

- {}で宣言し、その中にプロパティを、 ___プロパティ明:値___ の
