# 日報
- 無名関数について調べた
関数を変数の値として代入することができる  
const myfunc = function (name) {
  return `${name}さん、こんにちは！`;
}  
使う時は  
console.log(myfunc('ジョン'));  
のように変数名()で使うことができる
